First-time setup

Connect a Telegram bot to a local repository in about a minute.

Prerequisites

    1. Bot token
    2. Connect chat
    3. Project
    4. Done

    Message @BotFather in Telegram, send /newbot, and paste the API token it gives you.

    Open a chat with your new bot in Telegram and send it any message. The chat is detected automatically.

    Name the project and point it at a local Git repository.

    Absolute path. The directory must exist to save.

    🎉

    All set!

    Your bot is live. Open it in Telegram and send /start.

    Open the bot in Telegram Manage

    Once a project is added while the server is running, its bot goes live automatically.

    Prefer the full form? Use the Projects page.
